72V Sur-Ron Inspired eBike Build

A comprehensive guide to building a high-performance 72V version of the Sur-Ron Light Bee

Project Overview

The Sur-Ron Light Bee is one of the most popular lightweight electric motorcycles on the market. While the stock bike runs on a 60V system, this guide will help you build a more powerful 72V version with increased performance.

This upgrade will provide:

  • Increased top speed (potential 20-30% increase)
  • Improved torque and acceleration
  • Better hill-climbing ability
  • Faster response

Note: This is an advanced build that requires technical knowledge. Modifications may void warranties and impact legality in some areas.

    Building Process

    Step 1: Frame Preparation

    Start with a Sur-Ron compatible frame or modify an existing frame to accommodate the upgraded components. Ensure the frame can handle the increased power and stress.

    Step 2: Battery Configuration

    Assemble a 72V battery pack (20S) using high-quality cells. This may require a custom battery case and BMS system rated for the higher voltage.

    Step 3: Motor & Controller Installation

    Mount the upgraded motor and install the 72V controller. Ensure proper cooling for both components as they will generate more heat.

    Step 4: Transmission Setup

    Install the primary belt drive and secondary chain drive. Consider adjusting the gear ratio for your desired balance of torque and top speed.

    Step 5: Suspension & Brakes

    Install upgraded suspension components and ensure brakes are properly sized for the increased performance of the bike.

    Step 6: Electrical System

    Wire all electrical components including the display, lights, and throttle. Use appropriate gauge wiring for the higher current.

    Step 7: Testing & Calibration

    Carefully test all systems before riding. Configure controller settings for optimal performance and safety.
